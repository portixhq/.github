# PortixOne

A secure edge runtime that connects web applications to local hardware — printers, cash drawers, barcode scanners, scales, and more — through a unified developer API.

**Current status: MVP — Windows local printing only.** Everything else on the roadmap (cash drawer, scales, other capabilities, additional SDKs) is deliberately out of scope until this is validated with real developers.

Built in public, from Mexico, for a global developer audience.

## Start here

- **[portixone](https://github.com/portixhq/portixone)** — the monorepo. Runtime, JS SDK, and shared packages. Source of truth for development.
- **[portix.dev](https://github.com/portixhq/portix.dev)** — developer portal: docs, tutorials, examples, roadmap, changelog.
